# Olimpia Shakur • Flow Shop (mini-app)

Mini-app pronto para deploy na Vercel, com:
- Destaque: Copa do Mundo 2026 + Tech (Acessórios / Vestuário)
- Amazon Tech (Mais vendidos / Em alta)
- Hotmart Tech (+Quentes / +Queridos)
- Kiwify Tech
- Busca
- Acessibilidade: A- / A+ / 100% (salva no navegador)
- Tema: preto premium + dourado metalizado em detalhes + neon discreto (verde/lilás)

## Rodar local
```bash
npm install
npm run dev
```

## Atualizar links (1x por semana, sem retrabalho)
Edite `data/links.json`, faça commit e a Vercel redeploya automaticamente (se estiver ligado ao GitHub).

## Logo
Coloque sua logo em `public/logo.png`.
