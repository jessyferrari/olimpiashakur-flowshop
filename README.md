# Olimpia Shakur • Flow Shop (Mini‑App)

Mini‑app em Next.js (App Router) pronto para Vercel.

## Rodar local
```bash
npm install
npm run dev
```

## Publicar na Vercel
1. Suba este projeto no GitHub.
2. Na Vercel: **New Project → Import Git Repository → Deploy**.

## Ajustar links
Edite: `lib/links.ts`
